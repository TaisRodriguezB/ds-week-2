# Taller 3 - Análisis de comportamiento de compra

## Descripción
Este proyecto analiza el comportamiento de compra de usuarios en un dataset de e-commerce, con el objetivo de entender patrones de recurrencia, fidelidad y estructura de la canasta de compra.

## Objetivos
- Analizar el tiempo entre órdenes
- Evaluar patrones de compra por hora y día
- Estudiar la recurrencia de usuarios y productos
- Identificar productos clave en la canasta
- Analizar comportamiento de fidelidad


## Estructura del análisis
### Parte A: Análisis temporal
- Distribución de órdenes por hora y día
- Tiempo entre compras
### Parte B: Retención y productos
- Número de órdenes por usuario
- Top productos y principio de Pareto
### Parte C: Canasta y fidelidad
- Tamaño de la canasta
- Relación entre productos iniciales y recompra
- Proporción de recompra por producto
- Proporción de recompra por usuario
- Top productos añadidos primero


## Principales hallazgos
- Existe una alta variabilidad en el comportamiento de compra entre usuarios
- Los productos de consumo básico dominan las primeras posiciones del carrito
- La recompra es significativa tanto a nivel de producto como de usuario
- Se identifican patrones claros de fidelidad y recurrencia
- Algunos productos actúan como “anclas” en la canasta de compra
